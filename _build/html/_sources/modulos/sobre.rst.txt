Sobre o projeto
===============

O Sistema de Controle de Permanência tem como objetivo implementar os tópicos abordados nas disciplinas Desenvolvimento Web I e II.
Esta aplicação teve início em 2017 onde fora desenvolvida usando a linguagem Python juntamente com o Framework Django e obteve aprovação no exame de suficiência
da disciplina Desenvolvimento Web I.

Na atual versão da aplicação, foram feitas melhorias e adaptações para atender às necessidades do exame de suficiência da disciplina Desenvolvimento Web II,
implementação de testes automatizados, criação da documentação do projeto e a internacionalização são as principais mudanças nesta versão.
